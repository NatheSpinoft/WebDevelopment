HTML

Stands for Hyper Text Markup Language It is a container to hold everything from head, body to all contents within a page. Not to be confused with body which defines the document's body and is a container for all invisible contents.

Head

Elements containing meta information about the HTML page

Title

specifies the title for the page

H1 and p

H1 is the biggest font and subsequent numbers are of a lesser size.

p is to start a paragraph

HTML Attributes

All HTML elements have attributes. An attribute is an added paramented and specified in the start tag, and usually has a syntax similar to name="value". Similar to a href="www.site.com"
